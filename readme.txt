Ein kleiner Zahlensystemumrechner mit den folgenden Features

-Fehlerabfang wenn der User keine Nummer eingibt
-Fehlerabfang wenn der User die Nummer im falschen Zahlensystem eingibt
-Bemerkung, wenn man das selbe Zahlensystem zweimal ausgewählt hat
-Unterstützt Dezimal-, Binär-, Oktal- und Hexadezimalsystem
-Am Ende kann man das ganze wiederholen oder schließen
